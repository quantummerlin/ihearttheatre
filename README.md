## How to Use the AI Board

### **In VS Code with GitHub Copilot:**

**Step 1:** Open a file you want to work on (e.g., `index.html`)

**Step 2:** Open Copilot Chat

**Step 3:** Reference the specific agent:

```
Follow the instructions in agents/monetization_architect.md and audit the pricing structure in this file.
```

**Step 4:** Get specific recommendations based on that agent's framework

---

### **Recommended Sequencing:**

**For Landing Page Optimization:**
1. Conversion Architect ‚Üí Rewrite copy
2. Trust & Compliance ‚Üí Review claims
3. Product Differentiation ‚Üí Refine messaging

**For New Pricing Strategy:**

## Best Practices

### **DO:**
* Use one agent at a time (focused expertise)
* Reference the specific agent file
* Provide context about what you're working on
* Ask for specific outputs (not general advice)

### **DON'T:**
* Activate multiple agents at once (they'll blur)
* Skip the context (agents need to know what they're reviewing)
* Ignore the constraints (each agent has specific rules)
* Expect legal advice from the Compliance agent (consult a lawyer)

## Example Workflows

### **Workflow 1: Audit Current Pricing**
```
1. Open pricing.html
2. Copilot Chat: "Follow agents/monetization_architect.md and audit this pricing structure"
3. Get revenue leak identification
4. Implement recommended changes
5. Follow up: "Follow agents/data_feedback_agent.md and tell me how to track the impact of these pricing changes"
```

### **Workflow 2: Redesign Landing Page**

```
1. Open index.html
```
